# Pacôme Trousselle — Personal Site (static)

This is a lightweight static site (HTML/CSS/JS) designed to match the Jonathan Zong-style homepage layout, with a cinnabar accent color.

## Run locally (recommended)

Option A — Python (works on most machines):

```bash
cd pacome-site
python -m http.server 8000
```

Then open: http://localhost:8000

Option B — VSCode Live Server / any static server works too.

## Deploy to GitHub Pages

1. Create a GitHub repo (example: `pacome-site`).
2. Push these files to the repo root.
3. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` / root
4. Your site will be live on the GitHub Pages URL.

## CV

- Web CV: `cv.html`
- PDF download: `assets/pdf/cv.pdf` (placeholder)

Update the HTML CV content, then replace `assets/pdf/cv.pdf` with your real CV export.
